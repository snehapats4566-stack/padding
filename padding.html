Task 20: Rule Formation

  • Padding: internal spacing; use for hit-target and breathing room.
  • Margin: external spacing; use for spacing between components (or gap).
  • Flexbox: use when items need distribution, alignment, or responsive spacing; prefer gap over per-item margins.
  • box-sizing:border-box: makes width/height include padding/border — safer for components.
